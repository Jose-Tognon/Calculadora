# Calculadora

## Descrição

Este é meu primeiro projeto utilizando **Tkinter** para criar uma calculadora básica com interface gráfica em Python.

O objetivo do projeto é aprender sobre:
- Criação de interfaces gráficas (GUI) com Tkinter
- Manipulação de eventos e botões
- Estruturação de código Python para aplicações simples

## Funcionalidades

- Operações básicas: adição, subtração, multiplicação e divisão
- Interface simples e intuitiva
- Botões para números e operações
